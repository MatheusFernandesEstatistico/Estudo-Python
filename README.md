# Estudo de Python
Este repositório é dedicado ao estudo da linguagem Python, sem maiores aspirações, apenas aprendizado do 0. 

### Primeiro
Comece pelo arquivo *Curso Básico.ipynb* que são os módulos de um curso disponível no youtube, o link está no próprio código. Reveja a aula quantas vezes achar necessário e evite copiar e colar do código, exercite a "escrita do código".

### Segundo
É interessante começar a entender projetos básicos para se familiarizar com o funcionamento da linguagem. Existem muitos projetos comuns que ajudam a estudar, verifique as pastas de 'Proj_' que contém alguns exemplos.
